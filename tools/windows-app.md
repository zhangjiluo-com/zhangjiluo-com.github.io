### Windows Terminal

Windows 终端程序是一款新式、快速、高效、强大且高效的终端应用程序，适用于命令行工具和命令提示符，PowerShell 和 WSL 等 Shell 用户。主要功能包括多个选项卡、窗格、Unicode、和 UTF-8 字符支持，GPU 加速文本渲染引擎以及自定义主题、样式和配置。

这是一个[开源项目](https://github.com/microsoft/terminal)。

是微软开发的终端。如果无法安装，推荐使用 [Powershell](https://apps.microsoft.com/detail/9mz1snwt0n5d?hl=zh-CN&gl=CN) 代替。

## 代码编辑器（集成开发环境 IDE）

### Visual Studio Code (VS Code)

[Visual Studio Code](https://apps.microsoft.com/detail/xp9khm4bk9fz7q?hl=zh-CN&gl=CN) 是一款开源、免费、轻量且可扩展的 AI 代码编辑器，适用于使用任何编程语言和框架构建网页、桌面及移动应用。

Visual Studio Code 内置了对 Git 支持，能与 GitHub 实现强大集成，还配备了集成调试器，以及借助 IntelliSense 和人工智能驱动的 IntelliCode 实现的智能代码补全功能。在 Visual Studio Code 应用商店中，有超过 30,000 个扩展和主题，你可以通过它们自定义 Visual Studio Code 的功能和外观，以满足自己的需求、偏好和风格。

你可以使用 Visual Studio Code 构建各类应用，包括网页、桌面和移动应用。Visual Studio Code 原生支持 JavaScript 和 TypeScript，并提供了适用于多种语言编码的扩展，如 Python、Java、C/C++、C#、Go、Rust、PHP 等等。

[官网](https://code.visualstudio.com/)的定义：The open source AI code editor
